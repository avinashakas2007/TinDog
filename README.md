# TinDog

TinDog is a fun and interactive website designed to help dogs find friends and love nearby. It allows dog lovers to connect their pets with others through a simple and easy-to-use interface.

## Features
- **User-friendly design** with a responsive layout.
- **Dog profiles** and **matchmaking** for dogs.
- **Messaging** feature for interactions.
- **Multiple pricing plans**: Free, Premium, and Top-tier options.

## Technologies Used
- **HTML, CSS** for structure and styling.
- **Bootstrap** for responsive design.
- **Font Awesome** for icons.
